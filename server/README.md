# 远程作业发布系统 - 服务器端

## 简介

一个轻量级的远程作业发布系统服务器，使用 Node.js + Express + SQLite 构建，方便在学校环境中部署。

## 快速开始

### 1. 安装依赖

```bash
cd server
npm install
```

### 2. 启动服务器

```bash
# 开发模式（带热重载）
npm run dev

# 生产模式
npm start
```

服务器将在 http://localhost:3000 启动

### 3. 默认账户

- 用户名: `admin`
- 密码: `admin`

**注意**: 首次登录后请立即修改默认密码！
