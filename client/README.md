# 作业接收客户端

这是一个基于 Electron 的桌面客户端，用于接收老师发布的作业通知。

## 功能特性

- ✅ 实时接收作业通知（WebSocket）
- ✅ 系统托盘运行
- ✅ 新作业弹窗提醒
- ✅ 查看作业详情
- ✅ 标记已读/确认收到
- ✅ 支持按班级接收
- ✅ 离线消息同步

## 快速开始

### 1. 安装依赖

```bash
cd client
npm install
```

### 2. 开发模式运行

```bash
npm run dev
```

### 3. 构建应用

```bash
# 构建所有平台
npm run build

# 仅构建 Windows
npm run build:win

# 仅构建 macOS
npm run build:mac

# 仅构建 Linux
npm run build:linux
```

## 使用说明

### 首次使用

1. 打开应用后，点击"设置"标签
2. 输入服务器地址（如：`http://192.168.1.100:3000`）
3. 输入客户端名称（如："张三的电脑"）
4. 选择所属班级
5. 点击"保存设置"

### 接收作业

- 应用会在后台运行，收到新作业时显示系统通知
- 点击通知或应用图标查看作业详情
- 可以标记已读或确认收到

### 系统托盘

- 关闭窗口后应用会继续在托盘运行
- 右键托盘图标可以显示菜单
- 点击托盘图标可以显示/隐藏主窗口

## 配置说明

| 配置项 | 说明 | 示例 |
|--------|------|------|
| 服务器地址 | 后端服务器地址 | http://192.168.1.100:3000 |
| 客户端名称 | 显示名称 | 张三的电脑 |
| 所属班级 | 接收哪个班级的作业 | 高一(1)班 |

## 与课表软件集成

此客户端可以与 ElectronClassScheduleX 课表软件集成：

1. 在同一台电脑上运行两个应用
2. 作业客户端会在收到作业时显示通知
3. 可以在课表软件中添加作业客户端的快捷方式

## 技术栈

- Electron 28+
- WebSocket
- Axios
